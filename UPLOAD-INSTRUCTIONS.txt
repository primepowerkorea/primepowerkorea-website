PRIME POWER KOREA — GitHub Upload Instructions (V10)

1. Unzip this package.
2. Upload ALL files and folders to the ROOT of your GitHub repository.
3. Keep this exact structure:
   css/style.css
   js/main.js
   images/home/hero-warehouse.webp
   index.html
   products.html
   solutions.html
   brands.html
   about.html
   contact.html
4. Replace existing files when prompted.
5. Commit the changes and wait for GitHub Pages deployment to finish.
6. Refresh the website with Command + Shift + R on Mac.

V10 enlarges navigation, body copy, page headings, cards, lists, contact details and footer text across the entire website.
