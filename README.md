# Prime Power Korea Website

Static multi-page website prepared for GitHub Pages.

## Upload
Upload all files and folders to the repository root. Enable GitHub Pages from the main branch/root.

## Main files
- index.html
- products.html
- solutions.html
- brands.html
- about.html
- contact.html
- css/style.css
- js/main.js
- images/

All page text and local SVG illustrations are original to this project. The structure is inspired by standard industrial generator websites; no third-party website source code is included.
